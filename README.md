# Inspiration branch
This branch is used to store interesting implementations, which do not have anything to do directly with our code

### egn_state_estimator
This is the new, in-development state estimation from e-gnition, the TU Hamburg formula studen team.
It has not yet run on the car, but has a clean implementation.
It supports modular and asynchronous sensor inputs, so that sensors can be switched on and off, or be used at arbitrary frequencies.
